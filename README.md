# WeixinService
Python 微信个人订阅号开发

#功能
1.搞笑段子:段子<br />
2.星座运势:星座+运势(xxx座运势)<br />
3.手机归属地查询:手机+手机号码<br />
4.快递查询：快递+单号<br />
5.天气查询:城市+天气<br />
6.邮编查询:地名+邮编<br />
7.建议+内容(不少于五个字)<br />
